Guide d'utilisation

Ce que fait le dashboard

Ce dashboard centralise la comptabilite mensuelle de la Maison Abeille. Il presente les chiffres cles, les retrocessions par medecin, et les details de chaque activite.

OngletContenu
DashboardVue synthetique : 5 KPI principales (CA brut, dermato, chirurgiens, retrocessions, marge centre), 4 graphiques de repartition, synthese globale par source de revenus.
Fiches MedecinsFiche par medecin (cliquer pour voir le detail), analyse detaillee des actes et paiements, tableaux de retrocessions dermato et chirurgiens, recapitulatif paiement Dr Lellouch, activite jour par jour, traitements laser.
Donnees EditablesTableau interactif de toutes les lignes comptables. Chaque cellule est modifiable. Recherche et filtres par medecin ou categorie.
Statut / TarifsGrille tarifaire complete avec simulateur de retrocession. Acces protege par mot de passe.

Calculs automatiques

ActionResultat automatique
Modifier un Prix TTC dans Donnees EditablesLa valeur est sauvegardee en temps reel dans le navigateur (localStorage). Les totaux se recalculent.
Remplir Prix TTC + Taux dans Statut / Tarifs (formulaire)Calcul automatique du Prix HT, de la retrocession si TP, si TPMUT, du total centre et du total medecin.
Cocher TVA (20%) dans le simulateurLe Prix HT est automatiquement calcule : HT = TTC / 1.20
Renseigner TP et TPMUTRetro si TP = TTC - TP - (TTC x Taux). Retro si TPMUT = TTC - TPMUT - (TTC x Taux). Les valeurs negatives apparaissent en rouge.
Double-cliquer sur une cellule des tableaux DashboardEdition inline : la cellule devient un champ modifiable. Appuyer sur Entree pour valider, Echap pour annuler.

Fonctionnalites existantes

FonctionnaliteComment l'utiliser
Floutage des montantsPar defaut, tous les montants, graphiques et tableaux sont floutes. Cliquer sur "Verrouille" dans le header et entrer le mot de passe pour deverrouiller.
Exporter en CSVBouton "Exporter" dans le header (visible sur desktop). Telecharge un fichier CSV avec toutes les donnees.
ImprimerBouton "Imprimer" dans le header. Ouvre la boite de dialogue d'impression du navigateur.
Ajouter une ligneBouton "+ Ajouter" en haut de chaque tableau. Ajoute une ligne vide modifiable.
Supprimer une ligneCliquer sur le X a droite de chaque ligne dans les tableaux.
ReinitialiserBouton "Reinitialiser" dans l'onglet Donnees Editables. Remet toutes les donnees aux valeurs d'origine.
Filtrer / RechercherBarre de recherche et menus deroulants dans Donnees Editables et Statut / Tarifs.
Installer sur telephoneSur Android : bouton "Installer" dans le header. Sur iPhone : Partager > Sur l'ecran d'accueil. L'app s'ouvre ensuite en plein ecran.
Sauvegarde localeToutes les modifications sont sauvegardees automatiquement dans le navigateur (localStorage). Elles persistent entre les sessions.
